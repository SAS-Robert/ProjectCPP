Firmware changes

Feature overview

Version 2.1.16

  • Adapted to new hardware revision
  • Fuel gauge optimisation
  • Test impulse can not be switched off (setting the current to 0)
  • Bugfixes:
    • All 16 stimulation point can be used now

Version 1.4.4 (2016-07-20)

  • Device ID implemented
  • General
    • Added cmds
      • Smpt_get_main_status, response Smpt_get_main_status_ack
      • Smpt_get_stim_status, response Smpt_get_stim_status_ack
  • Anpassung der LEDs

Version 1.1.11 (2016-03-01)

  • Features
    • Stimulation
      • current adjustments (DAC)
      • Smpt_ll_channel_config_ack:
        • When no electrode occurs, electrode_error is set to Smpt_Channel_Undefined
      • EMG protection switches (Smpt_ll_emg_switches)
    • Data measurement (EMG)
      • Smpt_dm_init (new implemented parameters):
        • slot configuration,
        • sync with stimulation,
        • do_not_send_live_data (former send_live_data)
      • Smpt_dm_data (new implemented parameters):
        • stim_status (Mute-Signal)
  • Bugfixes
    • Electrode error in low level always returns electrode error red.
    • The device might stop sending packets when using high frequency stimulation and EMG.

Version 1.0.0 (2016-01-18)

  • Initial release
    • Low-level, Mid-level stimulation
    • Data measurement
