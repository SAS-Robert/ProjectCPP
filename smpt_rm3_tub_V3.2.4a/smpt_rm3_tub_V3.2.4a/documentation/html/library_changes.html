Library changes

Library changes

Version 3.2.4 (2017-03-17)

  • General
    • Removed
      • Command numbers of deprecated commands
      • Deprecated function structs
    • Changed
      • enum for Unknown_Cmd has a new value
    • New
      • function to get the version of the linked library (smpt_library_version())
      • New command number for general errors from the device (Cmd_General_Error)
  • Low-Level
    • Removed
      • smpt_ll_get_status
      • header documentation for demux (now in Low-level stimulation additional definitions and functions comments)
  • Mid-Level
    • Changed
      • ml_get_current_data: removed EMG and battery parameter
      • smpt_is_valid_ml_update() checks the parameters more restrictivly
  • Bugfixes
    • Linux: The flow control is deactivated before closing the serial port to prevent blocking.

Version 3.1.9 (2016-08-10)

  • General
    • Added
      • smpt_send_get_main_status()
      • smpt_send_get_stim_status()
  • Low-level
    • Smpt_ll_demux
      • Use the correct data types
  • documentation additions

Version 3.1.8 (2016-03-01)

  • Low level
    • Added
      • smpt_send_ll_emg_switches()
    • Removed
      • smpt_get_ll_ch_config_ack(), use smpt_get_ll_channel_config_ack() instead
      • smpt_send_ll_get_emg_reg(), smpt_send_ll_get_emg_data()
    • Type changes depending on the variable in General, Low- and Mid-Level
      • uint8_t -> bool
      • uint8_t -> Smpt_High_Voltage
      • uint8_t -> Smpt_Result
      • uint8_t -> Smpt_Channel
  • Data measurement
    • Renamed
      • Smpt_dm_init:
        • send_live_data -> do_not_send_live_data

Version 3.1.7 (2016-01-18)

  • Added demultiplexer support for low-level
  • Added data-measurement commands (for EMG measurement)
  • Added general commands (e.g. get battery, firmware-version)

Version 3.1.0

  • Bugfix 64-Bit Version

Version 3.0.4

  • First release for MiniStim (RehaStim2+ is not supported anymore)
  • Prefix changed sm2p -> smpt
  • Low level
    • Reduced number of channels 8 -> 4 (removed color information)
    • ll_channel_config: changed channel numbers back to old scheme before: 1-4, now: 0-3
    • EMG and Demux are not supported (will be added later)
    • Different changes for low-level cmds (parameter removed)
    • Changes in electrode error feedback
  • Added Mid-level cmds

Version 1.10

  • Bugfix release
